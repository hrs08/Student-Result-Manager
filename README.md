# Student-Result-Manager 🎓

This is a basic Python project I made for practice (under 30 lines).  
It uses file handling to store student result data in a `.txt` file.

## 📌 About

A simple Python-based command-line application to manage student results. This program allows users to input student data, calculate percentages, and store the results in a file.

## 📌 Features

- Add multiple students  
- Calculate total marks and percentage  
- Save results to a file (`results.txt`)  
- Supports multiple subjects per student  

## 🛠️ Tech Stack

- Python 3  

## 🚀 How It Works

1. Enter number of students  
2. Enter number of subjects  
3. Enter total marks per subject  
4. Input student names and marks  
5. Program calculates total and percentage  
6. Results are saved in `results.txt`  

## ▶️ How to Run

```bash
python student_result_manager.py
